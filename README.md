# AISA — Content Intelligence Dashboard

A Next.js 14 app that surfaces YouTube trending content by region and niche, built as a research tool for content creators and marketers.

## Features

- 📊 Dashboard with global YouTube trending videos
- 🗺️ Regional trends (US, UK, Europe)
- 🎯 Niche filtering (Marketing, AI, Finance, Gaming)
- 🌙 Dark SaaS-style UI with custom design system
- ⚡ Next.js 14 App Router + TypeScript + Tailwind CSS

## Getting Started

```bash
npm install
cp .env.example .env.local  # add your API keys
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) — redirects to `/dashboard`.

## Stack

Next.js 14 · TypeScript · Tailwind CSS · OpenAI API · YouTube Transcript API

## Status

Work in progress. Dashboard currently uses mocked data; YouTube API integration is the next milestone.
